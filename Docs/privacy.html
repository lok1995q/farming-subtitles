Farming Subtitles – 隱私權政策

Last updated: 2026-05-03

摘要：Farming Subtitles 是一款 Chrome 擴充功能，用於翻譯 YouTube 字幕與使用者主動選取的網頁文字。翻譯請求會傳送至使用者本機執行的 AI 服務（例如 http://127.0.0.1:1234），而不是由開發者代管的雲端翻譯伺服器處理。

Farming Subtitles is a Chrome extension that translates YouTube subtitles and user-selected webpage text using an AI service running on the user's own machine. This Privacy Policy explains how the extension handles data.

1. 簡介

Farming Subtitles 提供頁面文字翻譯功能，包括 YouTube 字幕翻譯，以及使用者主動選取之網頁文字翻譯。 本頁說明本擴充功能如何處理資料、資料用途，以及使用者可如何聯絡開發者。

2. 我們處理哪些資料

本擴充功能僅在提供翻譯功能所必要的範圍內處理以下資料：

  • 網站內容，例如使用者主動選取的文字內容。
  • YouTube 頁面中的字幕文字內容。
  • 設定與偏好，例如本機翻譯服務位址、模型名稱、系統提示詞與相關設定。

本擴充功能不會主動收集以下資料類型：

  • 姓名、地址、電子郵件地址、電話號碼或政府核發身分識別資料。
  • 付款、信用卡、銀行或財務資料。
  • 密碼、驗證碼、安全問題答案或其他登入驗證資訊。
  • GPS 定位資訊或精準位置資訊。
  • 完整瀏覽歷史清單。
  • 用於分析的滑鼠軌跡、鍵盤紀錄、點擊追蹤或捲動行為記錄。

3. 資料如何處理

  • 當使用者主動使用翻譯功能時，擴充功能會讀取需要翻譯的文字或字幕內容。
  • 這些內容會被傳送至使用者本機執行的 AI 服務，例如 http://127.0.0.1:1234，以取得翻譯結果。
  • 翻譯結果會回傳給擴充功能，並顯示在使用者當前頁面中。

本擴充功能不會下載或執行遠端程式碼。所有可執行程式碼皆封裝於擴充功能本身。

4. 資料儲存

  • 使用者設定（例如 server URL、模型名稱、提示詞與偏好設定）可能儲存在 Chrome 提供的儲存空間中，以便保留使用者設定。
  • 翻譯文字與翻譯結果可能暫時存在記憶體快取中，以減少重複翻譯請求並改善使用體驗。
  • 本擴充功能不會在開發者控制的雲端伺服器中建立集中式使用者資料庫。

5. 資料使用目的

本擴充功能處理資料的目的僅限於：

  • 提供 YouTube 字幕翻譯功能。
  • 提供使用者主動選取的網頁文字翻譯功能。
  • 將翻譯結果顯示於當前頁面。
  • 保存使用者偏好設定，以維持功能正常運作。

本擴充功能不會將資料用於：

  • 廣告投放或行銷用途。
  • 建立使用者個人檔案或跨網站追蹤。
  • 信用評分、貸款用途或與翻譯功能無關的目的。

6. 是否分享或出售資料

本擴充功能不會出售使用者資料，也不會將使用者資料轉移給第三方以供與本擴充功能核心用途無關之目的使用。

若使用者自行設定其他本機或自管服務端點，相關資料處理將受該使用者自行配置之服務環境影響。

7. 安全性

本擴充功能預設與使用者本機服務通訊，資料傳輸與保護程度將部分取決於使用者自身裝置與本機 AI 服務的設定。 使用者應僅使用信任的本機服務，並確保系統與相關軟體維持在安全狀態。

8. 使用者選擇

  • 使用者可停用或移除本擴充功能，以停止資料處理行為。
  • 使用者可修改本機服務位址或模型設定。
  • 若使用者不希望網頁內容被翻譯處理，請勿啟用翻譯功能或勿在頁面中選取文字後執行翻譯。

9. 政策更新

若本擴充功能未來的資料處理方式發生變更，本隱私權政策將一併更新，並以本頁面公布最新版本與更新日期。
