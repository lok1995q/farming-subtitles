Farming Subtitles – 隱私權政策
Last updated: 2026-05-03

1. 簡介
Farming Subtitles 是一款 Chrome 擴充功能，透過使用者本機執行的 AI 模型翻譯 YouTube 字幕與網頁中使用者主動選取的文字，並將翻譯結果顯示在當前頁面中。

本隱私權政策說明本擴充功能如何處理與保護使用者資料。

(English summary)
Farming Subtitles is a Chrome extension that translates YouTube captions and user-selected text using an AI model running on the user's own machine. This privacy policy explains how the extension handles your data.

2. 我們處理哪些資料
本擴充功能僅在提供翻譯功能所必須的情況下處理以下類型資料：

網站內容

使用者主動選取的網頁文字內容。

YouTube 影片頁面中的字幕文字內容。

設定與偏好

本機翻譯服務位址（例如 http://127.0.0.1:1234）。

使用的模型名稱與對應的系統提示詞。

擴充功能相關偏好設定（例如是否啟用翻譯、顯示方式等）。

我們不會刻意收集以下資料類型：

個人識別資訊（例如姓名、電子郵件、電話、地址、身分證號）。

健康資訊。

財務與付款資訊。

帳號密碼或驗證資訊。

個人通訊內容（例如電子郵件、即時通訊訊息）。

精準位置資訊（例如 GPS）。

使用者完整瀏覽歷史（我們不建立「造訪過哪些網站」的清單）。

點擊、滑鼠移動、捲動或按鍵行為的行為追蹤資料。

3. 資料如何處理與傳輸
本擴充功能會在使用者主動操作時（例如選取文字並啟用翻譯、觀看影片時啟用字幕翻譯），讀取當前頁面中需要翻譯的文字或字幕。

讀取到的文字會被傳送至使用者本機執行的 AI 服務（預設為 http://127.0.0.1:1234）進行翻譯。

翻譯結果會回傳至擴充功能，並顯示在使用者當前頁面上（例如插入在選取文字附近，或覆蓋在 YouTube 字幕上方）。

所有可執行程式碼（JavaScript 等）皆封裝於擴充功能套件中，本擴充功能不會從遠端載入或執行外部託管程式碼。

4. 資料儲存位置與期間
設定與偏好（例如本機服務位址、模型名稱、提示詞與相關選項）會儲存在 Chrome 提供的 chrome.storage 中，用於在未來使用時保留設定。

翻譯用的文字與翻譯結果可能會暫存在記憶體中（例如快取近期字幕或句子，以避免重複請求），不會被長期儲存為獨立的使用者資料檔案。

我們不會在伺服器端（雲端）為此擴充功能建立使用者帳戶，也不會在我們控制的伺服器上集中儲存使用者資料。

5. 是否與第三方分享資料
本擴充功能設計為與使用者本機執行的 AI 服務通訊，預設為 127.0.0.1，該服務由使用者自行安裝與控制。

我們不會將使用者資料出售或轉移給任何第三方服務，除非為了提供使用者明確要求的核心功能，且符合 Chrome Web Store 的 Limited Use 要求。

我們不會將使用者資料用於與本擴充功能單一用途無關的目的，例如廣告投放、第三方分析或建立使用者個人檔案。

6. 使用目的
我們僅將上述資料用於以下用途：

提供文字與字幕翻譯功能。

顯示翻譯結果於使用者當前頁面。

儲存與套用使用者的偏好設定（例如翻譯開關、模型選擇等）。

我們不會將資料用於：

信用評等或貸款決策。

與翻譯功能無關的行銷或廣告。

建立跨網站的使用者行為追蹤檔案。

7. 安全性
所有翻譯請求預期在使用者本機網路範圍內傳輸（例如 http://127.0.0.1:1234），實際連線安全性與保護機制也取決於使用者所配置的本機服務。

我們建議使用者僅使用信任的本機 AI 服務，並確保相關軟體和系統已適當保護。

8. 未成年使用者
本擴充功能並非專為未成年人設計或行銷。我們不會刻意識別或區分未成年使用者，也不會要求填寫年齡或身份資訊。

9. 政策變更
若未來擴充功能新增會影響隱私的功能或資料處理方式，我們會更新本隱私權政策，並在 Chrome Web Store 揭露最新資訊。

更新後的隱私權政策會標註生效日期，並公開於相同網址。
