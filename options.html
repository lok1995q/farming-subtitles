LM Studio 設定 / LM Studio Settings

LM Studio 本機 server 的地址，預設為 http://127.0.0.1:1234

選擇預設模型，或選 Other 自訂 model id

每個模型可以有各自的 Prompt，切換模型時自動載入 / Each model can have its own prompt

選擇後會自動填入上方 Prompt 欄位，記得點儲存 / Selecting will fill the prompt above, remember to save

請輸入 LM Studio API 可識別的 model 名稱

📖 快速設定指南 / Quick Setup Guide
  1. 下載並安裝 LM Studio
  2. 在 LM Studio 載入你要使用的模型
  3. 進入 Developer 頁，啟動 server
  4. 回到這裡選擇模型後，點「測試連線」確認設定正確
